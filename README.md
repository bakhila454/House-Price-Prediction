# House-Price-Prediction# House Price Prediction using Linear Regression

## Description
This project predicts house prices using:
- Square footage
- Number of bedrooms
- Number of bathrooms

## Tools Used
- Python
- Pandas
- Scikit-learn
- Google Colab

## Output
- Predicted house prices
- Mean Squared Error

## Conclusion
The model successfully predicts house prices using linear regression.
